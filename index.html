Getting started with Core Hunter

Core Hunter is implemented in Java and can be executed from R or using the graphical interface. In both cases you will need to have installed a Java Runtime Environment (JRE) version 8 or later.

R Package

The R package corehunter is available on CRAN and can be installed with

> install.packages("corehunter")

Graphical interface

The graphical interface of Core Hunter is under development. More information will be posted here soon.
